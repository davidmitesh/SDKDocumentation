    Including back-end js sdk in your nodejs project

    npm i hamro_service_message_sdk --save

const {sendNotification}=require('hamro_service_message_sdk')

    Create a file .env in the root project directory where index.js of nodejs server is hosted. The file format should be :

    MINI_APP_ID=**INSERT YOUR MINI-APP-ID HERE**
MINI_APP_SECRET_KEY=**INSERT YOUR SECRET KEY HERE**
MINI_APP_PUSH_KEY=**INSERT YOUR PUSH KEY HERE**
GRPC_TARGET=**INSERT GRPC TARGET FOR HAMROPATRO SERVICE MESSAGE**

    After filling in the values, the file should look like:

    MINI_APP_ID=6caff600-b7ca-11eb-baa9-0242ac1200aa
MINI_APP_SECRET_KEY=GK7zYe#GR#Rma@l6oYtWoqcoSD%RxkmFseiYFj0J3dlzzysGWAump!p%VM#e
MINI_APP_PUSH_KEY=GK7zYe#GR#Rma@l6oYtWddccoSD%RxkmFseiYFj0J3dlzzysGWAump!p%VM#e
##This is dev Grpc target
GRPC_TARGET=mini-app-grpc-dev.hamrostack.com
